## Dataset Information

Dataset ini berisi transaksi penjualan kedai kopi dengan sekitar 150.000 baris. 
Kumpulan data yang digunakan dalam proyek ini disediakan sebagai bagian dari program pelatihan Analis Data TutorMy.

Main columns:
id_transaksi
tanggal_transaksi  
waktu_transaksi 
jumlah_transaksi
id_toko
lokasi_toko 
id_produk 
harga_satuan 
kategori_produk
tipe_produk 
detail_produk
